## Technologies used

This project follows a custom structure that combines the following:

- HTML5
- CSS3
- JavaScript (ES6)
- Swiper
- Custom fetch wrapper for HTTP-methods
- Vite

The project utilizes the Movie Database API for fetching movie information.
